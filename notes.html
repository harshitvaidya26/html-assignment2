HTML Tags and Descriptions

  1. <h1> to <h6> : Heading tags, used to define headings of different sizes.
  2. <p> : Paragraph tag, used to define a block of text.
  3. <a> : Anchor tag, used to create hyperlinks.
  4. <img> : Image tag, used to embed images in a webpage.
  5. <ul> and <li> : Unordered list and list item tags, used to create bullet-point lists.
